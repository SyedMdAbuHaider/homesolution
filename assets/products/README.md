# Product images

Drop product photos here, e.g. product1.webp, product2.webp ...

Then reference them in data/products.json under each product's "image" field:

  "image": "assets/products/product1.webp"

Recommended: .webp, roughly 4:3.2 aspect ratio, under 150KB each for fast loading.
If an image is missing, the site automatically falls back to a category icon —
nothing breaks.
